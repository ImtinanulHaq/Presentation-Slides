# Presentation Tools Frontend

React-based frontend for Presentation Tools application.

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm start
```

Runs the app in development mode. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Build

```bash
npm run build
```

Builds the app for production to the `build` folder.

### Technologies Used
- React 18.2.0
- React Router 6.20.0
- Axios 1.6.2
- Ant Design 5.11.0
- Zustand 4.4.1
- React Quill 2.0.0
